== Description ==

Adds a sidebar widget that uses the posts's featured image as a link to open a lightbox gallery.

Integrates with lightbox gallery

== Installation ==


1. Upload `gt-lightbox-gallery-launcher` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag widget onto sidebar
